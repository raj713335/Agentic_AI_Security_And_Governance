# AI System Card

## System Name
Support Operations Agent

## Purpose
Assist support teams with documentation search, ticket creation, customer communication drafts, and reviewed operational actions.

## Agent Capabilities
- Search public support docs
- Create support tickets
- Draft customer emails
- Request record deletion

## Top Risks
- Prompt injection
- Goal hijacking
- Tool misuse
- PII leakage
- MCP server supply chain risk
- Multi-agent cascading failure

## Core Controls
- Authentication
- Scoped authorization
- Policy gates
- Human review
- PII redaction
- Output validation
- Audit logs
- Anomaly detection
- Kill switches
